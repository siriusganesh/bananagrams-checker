# Bananagrams Word Checker

Live site: https://siriusganesh.github.io/bananagrams-checker/

Type letters into the box and hit **check**. The tool will tell you:

1. Whether your input is a legal Bananagrams word.
2. Every legal word you can build from those letters — full anagrams first, then sub-anagrams grouped by length.

Dictionary: ~279k words (a Collins/SOWPODS-derived superset that includes every TWL Scrabble word). For casual play this is more permissive than strict North American TWL — words like QI, ZA, OK all validate.
